# Zero to Dapp Devconnect Argentina

This is the repo for Zero to Dapp which is a workshop with seven sections each section will have at least one branch.  The
1. Remix
2. The Graph
3. Celo MCP Composer
4. Chainlink
5. ENS
6. Uniswap
7. Self Protocol

When there is just one branch for a section, it will be the complete version.  When there are two branches, the first one is for starting and the second is will contain the complete files for that section.
